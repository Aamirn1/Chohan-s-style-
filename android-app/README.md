# Chohan's Style Hub - Android App

Native Android WebView app that loads the live website for a pixel-perfect mobile experience.

## Build
This app is built automatically via GitHub Actions. See `.github/workflows/main.yml`.

## Manual build
```bash
cd android-app
./gradlew assembleDebug
# APK at app/build/outputs/apk/debug/app-debug.apk
```
# Build trigger: bottom nav always visible in WebView
